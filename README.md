# shawarmys

Shawarmys is a small containerized web application (React frontend + backend services) used in this hackathon workspace. The repository contains a `client` (frontend) and `server` (API/services) directory, plus Docker compose files to run everything locally.

**Quick Start (recommended)**

- **Prerequisites:** Docker and Docker Compose installed.
- **Start:**
  **_Node_**
  npm install
  npm run start

  **_Python_**
  cd server/misc
  python3.12 -m venv venv
  pip install -r requirements.txt
